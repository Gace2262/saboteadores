# Community Rules

El Archivo Público acepta contenido simbólico, ficticio y seguro.

## Permitido

- Cartas, bosses, campañas, seeds y desafíos.
- Humor negro centrado en patrones mentales, no personas reales.
- Warnings claros para contenido intenso.
- Contenido original o con permiso.

## No permitido

- Odio contra grupos reales.
- Amenazas reales.
- Autolesión explícita.
- Malware o scripts.
- JavaScript externo.
- HTML arbitrario.
- Pornografía explícita.
- Contenido ilegal.
- Assets robados o con copyright no autorizado.
- Monetización engañosa.

## Warnings

Declara sellos cuando corresponda:

- flashing
- loud audio
- psychological horror
- intense corruption
- difficult gameplay
- burnout themes

Mensaje oficial: el Tribunal puede ser dramático, pero el archivo no ejecuta cuchillos.
