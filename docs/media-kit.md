# Media kit

## Descripcion corta

Un juego de cartas psicologico y cinematografico donde enfrentas Saboteadores Mentales dentro de un tribunal vivo construido por mecanismos emocionales.

## Descripcion larga

Saboteadores Mentales: Habitantes Invisibles combina deckbuilding tactico, bosses colosales, cartas evolucionables, musica dinamica y humor negro ceremonial dentro de una mente que convierte patrones automaticos en tiranos con martillo. No estas salvando un reino externo: estas apelando tu propia sentencia interna.

## Generos

- Deckbuilder
- Roguelike
- Psychological Strategy
- Narrative Card Game
- Dark Fantasy Mental Opera

## Features

- Combate tactico de cartas.
- Bosses multi-fase.
- Cartas con evolucion, corrupcion y despertares.
- Campana narrativa.
- Musica dinamica y cinematicas.
- Tribunal Vivo con eventos emergentes.
- Mods locales y estructura para comunidad.
- Humor negro centrado en patrones mentales.
