# Branding guide

## Identidad

**Nombre:** Saboteadores Mentales  
**Subtitulo:** Habitantes Invisibles  
**Posicionamiento:** Una opera psicologica de cartas con estetica de tribunal mental epico.

El juego debe sentirse como un grimorio moderno, tarot distorsionado, tribunal gotico y album coleccionable grotesco dirigido por una banda de metal sinfonico sin supervision emocional.

## Logo

Versiones oficiales:

- Full logo: titulo completo + subtitulo.
- Compact logo: solo "Saboteadores Mentales".
- Icon only: sello judicial fracturado con ojo oculto.
- Monochrome: uso en documentos y prensa.
- Cinematic animated logo: humo, cadenas, sello, grietas y ojo dorado.

## Colores

- Negro carbon `#070707`: fondos, tribunal, vacio.
- Dorado viejo `#d6aa45`: sellos, logo, rarezas.
- Rojo ansiedad `#b91c1c`: estres, alertas, bosses.
- Azul culpa `#5b8fb9`: culpa, lluvia mental.
- Violeta mental `#7c3aed`: corrupcion, ruido mental.
- Gris humo `#8d8a91`: niebla, UI secundaria.
- Blanco catarsis `#f8fafc`: liberacion y despertares.

## Tipografia

- Titulos: gotico moderno, mayusculas, peso alto.
- Subtitulos: limpios, espaciado amplio.
- Gameplay: legible, contraste alto.
- Impactos: cinematograficos, breves y claros.

## Uso visual

- Usar humo suave, grietas luminosas, cadenas sutiles y sellos judiciales.
- Evitar fondos genericos o fantasia medieval sin contexto psicologico.
- Las cartas y materiales deben comunicar emociones, mecanismos y juicio interno.
- Crear versiones estaticas para fotosensibilidad.

## Facciones

- El Juez: oro oscuro, martillo, ojo, vitrales.
- Controlador: geometria, formularios, cadenas ordenadas.
- Inquieto: humo rapido, luces rotas, caos.
- Hipervigilante: alarmas, radares, pulsos rojos.
- Reservado: azul oscuro, vacio, niebla.
- Conciencia: blanco catarsis, oro vivo, cadenas oxidadas.

## Exportaciones

Preparar PNG, WEBP y SVG para logos. Usar composiciones separadas para Steam, redes, posters y trailer.
