# Tester Guide - Saboteadores Mentales

## Version

Jugar `0.1.0-public-demo`.

## Como abrir la demo

1. Abrir la URL publica.
2. Entrar a `/landing` para leer el resumen.
3. Pulsar **Jugar demo web** o abrir `/demo`.

## Duracion esperada

15 a 25 minutos.

## Que probar

1. Iniciar demo.
2. Cambiar ajustes de audio o accesibilidad.
3. Elegir mazo.
4. Jugar combate.
5. Probar ganar o perder.
6. Ver cambios de fase del boss.
7. Llegar a pantalla final.
8. Reiniciar demo.
9. Abrir `/feedback`.
10. Guardar y exportar feedback.

## Que reportar

- No se puede iniciar demo.
- No se puede elegir mazo.
- El combate no termina.
- La IA no juega.
- El audio queda pegado.
- La UI se rompe en mobile.
- Textos confusos.
- Dificultad del boss.
- Parte mas entretenida.
- Parte mas confusa.

## Capturas utiles

- Pantalla inicial.
- Seleccion de mazo.
- Combate con cartas en mano.
- Boss en fase 2 o fase 3.
- Pantalla de victoria o derrota.
- Cualquier error visual.

## Feedback

Usar `/feedback`. No se piden datos personales. El formulario guarda localmente y exporta JSON.
