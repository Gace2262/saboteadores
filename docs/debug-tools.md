# Debug Tools

Rutas:

- `/debug`
- `/debug/saves`
- `/debug/diagnostics`

Funciones:
- agregar monedas.
- desbloquear carta.
- simular sobre.
- activar boss/cinematica.
- activar Tribunal Vivo.
- exportar/importar/resetear save.
- ver errores recientes.
- ejecutar diagnostico interno.

No hay telemetria externa. Todo queda en localStorage.
